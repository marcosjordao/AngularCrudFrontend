# CRUD Single Page Application - Frontend

CRUD Angular single page application.
 
## Coded with:
 * **[Angular (v 7.3.3)](https://angular.io/)**
 * [TypeScript](https://www.typescriptlang.org/)
 * [Bootstrap (v 4.3.1)](http://getbootstrap.com/)
 
## How to run
```
cd WebSPA
npm install
ng serve --open
```
